# Hello Toast Android Application

## Description
Hello Toast is a simple Android application developed in Java with Android Studio.
It helps learn the basics of Android development such as using buttons, displaying Toast messages, and managing a counter.

The user can:
- display a Toast message
- increment a counter
- see the counter value updated on screen

## Features
- Display a Toast message
- Increment a counter
- Update a TextView
- Interaction between XML and Java

## Application Interface
The application contains:
- **SHOW TOAST Button**: displays a Toast message
- **Yellow Center Area**: displays the counter value
- **Say hhiiiiii! Button**: displays a welcome message
- **INCREASE up up Button**: increments the counter

## Technologies Used
- Java
- Android Studio
- Android SDK
- XML Layout

## How It Works

### SHOW TOAST Button
Displays a Toast message: allo allo hello toast


### Say hhiiiiii! Button
Displays a Toast message:wiiii haaaaallooo hry?


### INCREASE up up Button
Increases the counter value by 1 each time the user clicks.

### Counter Display
The counter value is displayed in a TextView in the center of the screen.

## Main Code
MainActivity.java handles:
- button clicks
- counter increment
- Toast message display

## How to Run the Application
1. Open the project in Android Studio
2. Wait for Gradle to sync
3. Launch an Android emulator or connect a phone
4. Click Run

## Learning Objectives
This application helps you learn:
- Android layouts
- click events
- using Toast
- updating the user interface
